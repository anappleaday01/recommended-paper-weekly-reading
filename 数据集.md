
# **HBN 单个受试者全维度数据采集总览**

每个参与HBN研究的儿童或青少年（5-21岁）都接受了为期四个会话、约12-14小时的综合评估，生成以下**八大类数据**：

---

## **一、行为与心理评估**

### **1. 临床诊断评估**
- **半结构化诊断访谈**：使用儿童版情感障碍与精神分裂症计划表（KSADS-COMP）进行访谈，生成DSM-5诊断
- **补充诊断评估**（根据临床需要）：
  - **自闭症谱系障碍**：ADOS-2、ADI-R
  - **语言障碍**：CELF-5、TLC-4、EVT-2、PPVT-4、GFTA-3
  - **强迫症**：Y-BOCS
  - **抽动障碍**：Y-GTSS
  - **适应性行为**：Vineland量表

### **2. 智力与学业能力**
- **智力测试**：
  - 5岁或IQ<70：KBIT
  - 6-17岁：WISC-V
  - 18岁以上：WAIS-IV
- **学业成就测试**：WIAT-III（6岁以上）

### **3. 语言能力筛查与评估**
- **筛查工具**：CELF-5筛查版、GFTA发音测试、CTOPP-2、TOWRE-2
- **完整语言评估**：如筛查未通过，由语言病理师执行完整评估

### **4. 自评与他评问卷**
#### **A. 精神病理学与行为**
- **儿童行为量表**：CBCL/YSR/ASR（内化、外化问题）
- **焦虑**：儿童焦虑相关情绪障碍筛查量表 (SCARED)
- **抑郁**：情绪与感受问卷 (MFQ)
- **ADHD**：E-SWAN、ADHD评定量表
- **自闭症特征**：ASRQ、SRS-2
- **自杀风险**：哥伦比亚自杀严重程度评定量表 (C-SSRS)

#### **B. 功能损害与优势**
- 哥伦比亚损害量表 (CIS)
- 世界卫生组织残疾评估表 (WHODAS)
- 社交能力量表 (SAS)

#### **C. 家庭、压力与环境**
- 家庭历史：研究用诊断标准家庭史
- 父母压力：父母压力指数第四版 (PSI-IV)
- 童年不良经历：童年不良经历量表 (ACES)
- 家庭冲突：儿童对人际冲突感知量表 (CPIC)

#### **D. 生活方式与健康**
- 体力活动：体力活动问卷 (PAQ-C/A)
- 营养：食物频率问卷 (FFQ)
- 物质使用：AUDIT、IAT等
- 睡眠：儿童睡眠障碍量表 (SDSC)

---

## **二、生理、体能与生物样本信息**

### **1. 生理指标**
- **体测**：身高、体重、腰围、握力、坐位体前屈
- **心血管**：静息心率、血压
- **体成分**：生物电阻抗分析测量体脂率、去脂体重

### **2. 体能测试**
- **FitnessGram**：评估有氧能力、肌肉力量/耐力、柔韧性

### **3. 生物样本**
- **唾液**：用于全基因组遗传分析
- **血液**（部分受试者）：内分泌、免疫、代谢组学和遗传学分析
- **尿液**：毒理学筛查和妊娠测试
- **乳牙**：重建产前及儿童早期金属暴露（铅、锰等）时间序列
- **头发**（计划中）：评估近期金属暴露

---

## **三、多模态脑成像 (MRI) 信息**

### **1. 结构磁共振成像**
- **高分辨率T1加权像**：脑形态计量、皮层厚度、表面积分析
- **T2加权像/T2-FLAIR**：区分组织类型、检测病变
- **磁化转移成像**：评估髓鞘完整性

### **2. 功能磁共振成像**
- **静息态fMRI**：构建全脑功能连接网络
- **自然观看fMRI**：观看《Despicable Me》和《The Present》短片，研究自然刺激下的脑活动

### **3. 扩散磁共振成像**
- **扩散峰度成像 (DKI)**：64个方向，刻画白质微结构复杂性

### **4. 定量成像**
- **DESPOT T1/T2 mapping**：定量测量组织弛豫时间
- **腹部成像T1W**：评估内脏脂肪

---

## **四、高密度脑电图 (EEG) 与眼动信息**

### **1. EEG范式**（包含原始`.mff`及预处理`.csv/.mat`数据）
- **静息态**：闭眼/睁眼
- **认知任务态**：
  - **SAIIT 2AFC**：注意维持与决策
  - **环绕抑制**：视觉知觉抑制
  - **视觉学习**：模式学习与记忆
  - **WISC加工速度**：行为测试与EEG同步
  - **其他任务**：Flanker、Go/No-Go、N-back、情绪面孔等
- **自然观看**：观看《The Present》等短片（与fMRI任务匹配）

### **2. 同步采集**
- **高密度EEG**：128通道，500Hz采样
- **高精度眼动追踪**：记录注视点、瞳孔直径，与EEG时间锁定
  - 空间分辨率：0.1°
  - 采样率：120Hz
  - 用于校准与注视点分析

### **3. EEG数据集结构**
```
NDARAA075AMK/
├── Behavioral/                # 行为数据（任务表现、按键反应）
├── EEG/
│   ├── preprocessed/         # 预处理后的EEG数据
│   ├── raw/                  # 原始EEG数据
│   └── mff_format/           # 原始EEG专有格式（EGI系统MFF文件）
└── Eyetracking/              # 同步眼动数据
```

### **4. EEG文件对应关系表**
| **文件名** | **实验范式** | **数据类型** | **说明** |
|-----------|-------------|-------------|----------|
| `RestingState_*` | 静息态 | 连续脑电数据 | 用于计算静息态功能连接、频谱特征 |
| `SAIIT_2AFC_*` | SAIIT二选一决策 | 注意与决策任务 | 视觉/听觉决策任务 |
| `SurroundSupp_*` | 环绕抑制 | 视觉知觉抑制 | 研究视觉皮层抑制机制 |
| `Video1/2/3_*` | 视频观看 | 自然观看EEG | 与fMRI自然观看任务对应 |
| `vis_learn_*` | 视觉学习 | 学习与记忆 | 模式学习或记忆编码/提取 |
| `WISC_ProcSpeed_*` | 韦氏加工速度 | 认知加工速度 | 与WISC分测验对应的EEG任务 |

### **5. EEG BIDS标准格式**
成像数据以BIDS格式存储，结构如下：
```
sub-<ID>/
  ├── eeg/
  │   ├── sub-<ID>_task-<TaskName>_eeg.set    # EEGLAB格式
  │   ├── sub-<ID>_task-<TaskName>_eeg.fdt    # 数据文件
  │   └── sub-<ID>_task-<TaskName>_events.tsv # 事件文件
  └── sub-<ID>_sessions.tsv                   # 会话信息
```

**优势**：标准化命名、丰富元数据，便于跨研究、跨工具数据交换

### **6. 眼动数据**
- **格式**：`.idf` (SMI专有) 和 `.txt` (通用文本)
- **内容**：注视点坐标、瞳孔直径、眨眼事件，与EEG时间对齐
- **用途**：排除眼动伪迹、分析注视行为与脑电活动关联

### **7. 行为数据**
- **位置**：`Behavioral/`目录
- **内容**：任务中的按键反应、反应时、正确率
- **用途**：作为EEG分析的协变量或分组依据

## **五、数字行为与语音信息**

### **1. 语音录音**
- **日常评估录音**：全程录制临床访谈、认知测试中的对话
- **叙事任务录音**：MRI扫描后，复述观看的短片并回答问题

### **2. 视频录制**
- 叙事任务时同步录制**面部表情和上半身肢体语言**

### **3. 活动监测**
- 佩戴腕式ActiGraph活动记录仪**连续一个月**
- 监测**日常活动模式与睡眠节律**

### **4. 语音采集具体情境**
#### **A. 临床诊断访谈（KSADS-COMP）**
- **内容**：持照临床医生进行的半结构化访谈
- **价值**：记录"症状报告"行为本身，分析言语特征与症状严重度关系

#### **B. 认知与学业能力测试**
- **内容**：韦氏智力量表、学业成就测试等
- **价值**：关联认知表现得分与达成过程（言语特征）

#### **C. 语言能力专项评估**
- **内容**：CELF、GFTA等标准化测试
- **价值**：提供高保真语音样本，精确分析发音错误模式

#### **D. 问卷调查补充访谈**
- **内容**：问卷条目的澄清、追问
- **价值**：捕捉更自发、少结构化的言语特征

---

## **六、人口学、社会经济学与纵向追踪**

### **1. 人口学信息**
- 年龄、性别、利手、种族、邮编

### **2. 社会经济学**
- 父母教育、职业、家庭收入（Barratt简化社会地位测量）

### **3. 纵向追踪**
- **季度心理健康报告**：通过问卷跟踪症状变化
- **服务使用评估**：记录心理健康服务使用情况

---

## **七、数据整合与独特价值**

| **维度** | **数据交集示例** |
|----------|------------------|
| **神经-行为** | EEG N-back数据 + WISC工作记忆分数 + fMRI额顶网络连接 |
| **基因-脑-环境** | 遗传数据 + 牙齿金属暴露 + 皮层厚度 + CBCL外化问题分 |
| **动态监测** | 活动监测睡眠效率 + 季度焦虑分数 + 静息态fMRI杏仁核连接 |
| **跨模态验证** | 自然观看EEG反应 + 同一任务fMRI反应 + 眼动注视模式 |
| **语音生物标志物** | 访谈录音声学特征 + SCARED焦虑总分 + EEG额叶不对称性 |

---

## **总结**

HBN为每一位受试者构建了一个 **"深表型"数据宇宙**，覆盖了从**分子（遗传）、细胞（环境暴露）、环路（EEG/fMRI）、认知（测试）、行为（问卷）、环境（家庭）到动态监测（活动、语音）** 的完整链条。

这使其成为全球首个旨在通过**数据驱动**方式，重新定义儿童精神健康与学习障碍的**跨诊断、多模态、纵向大型开放资源**。
